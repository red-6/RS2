A) HTML Structure - top

This theme is a responsive layout, mainly with two columns, except full-width page, and some others. All of the information within the main content area is a div with an class of "content", but actual content goes usually within a div with an class of "left-side". The sidebar's content is within a div with an class of "right-side". The general template structure is mostly the same throughout the template. Here is the general structure.



If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

	body { font: 12px/18px Arial, serif; color: #76452f }
			
	a { color: #a65600; text-decoration: underline; }

	a:hover { text-decoration: none; }

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

I.E.

	body .wrapper .item { font: 12px/18px Arial, serif; color: #76452f }

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

B) CSS Files and Structure - top

There is only one CSS file for main page styling called main-stylesheet.css - it includes element resets and all of the page structure styling, fonts, colors, block elements, etc.

It's structured by sections like this:

	/* 0.0 Reset */
	
	CSS code
	

	/* 1.0 Layout Structure */
	
	CSS code
	
	
	/* 2.0 Main Header */
	
	CSS code
	
	/* 3.0 Primary Horizontal Menu */
	
	CSS code
	
	
	etc, etc.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

C) JavaScript - top

This theme imports six Javascript files.

  1. jquery-latest.min.js - standart jQuery
  2. demo-settings.js - demo settings
  3. lightbox.js - lightbox js file
  4. theme-scripts.js - theme scripts
  5. museo.font.js - custom font for Cufon

D) PSD Files - top

There are one .psd file for each HTML template. They all have nicely structured layers with groups and titles and guidlines.
